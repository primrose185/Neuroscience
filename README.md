# Frontend for Neuroscience Website

## Setup

1. Install [Yarn](https://classic.yarnpkg.com/en/docs/install/)
2. Install dependencies:
   ```bash
   cd frontend
   yarn install
   ```
3. Run the dev server:
   ```bash
   yarn dev
   ```
4. Run tests:
   ```bash
   yarn test
   ```

## Features
- Vue 3 + Vite + TypeScript
- Tailwind CSS
- shadcn-vue UI components
- Vitest for testing
- Example API integration
